1. Поместите файлы DD_belatedPNG_0.0.8a-min.js, IE8.js, ie7-squish.js  в папку js своего сайта
2. Поместите файл blank.gif в корень сайта
3. В функции fix() пропишите селекторы тех элементов, которым нужна поддержка прозрачности.
4. Подключите этот код в обасть head всех страниц сайта:

<!--[if lt IE 7]>
<script type="text/javascript" src="js/DD_belatedPNG_0.0.8a-min.js"></script>
<script>
  DD_belatedPNG.fix('#new,#bird,#button');
</script>
 <script src="js/IE8.js" type="text/javascript"></script>
 <script src="js/ie7-squish.js" type="text/javascript"></script>
<![endif]-->
